# 대전동화중학교 생활기록부 도우미 (2026학년도)

교사가 **학년·반을 고르고 키워드만 입력하면**, 학교의 실제 학사일정·창의적 체험활동·동아리·교과
자료에 비추어 **200~300자 생활기록부 초안**을 자동으로 만들어 주는 웹앱입니다.
학급이 함께한 활동(예: 마약류 예방교육)은 **버튼 한 번으로 반 전체 초안**이 학생마다 다른 표현으로 채워집니다.

> 인터넷 없이도 동작합니다. 모든 학생 명단과 학교 활동 데이터가 앱 안에 들어 있습니다.

---

## 실행 방법

### 가장 쉬운 방법
`index.html` 파일을 **더블클릭**하면 기본 브라우저(크롬/엣지)에서 바로 열립니다.

### 권장 방법 (복사 기능까지 안정적으로)
`실행.bat` 을 더블클릭 → 자동으로 로컬 서버가 켜지고 브라우저가 열립니다.
(Python이 설치되어 있어야 합니다. 닫을 때는 검은 명령창을 닫으면 됩니다.)

---

## 사용 순서

1. **학년 → 반** 클릭 → 그 반 학생이 **표**로 나타남
2. 상단에서 작성할 **영역**과 세부 **항목**(활동/과목) 선택
   - 자율활동 · 동아리활동 · 진로활동 · 봉사활동 · 행동특성 및 종합의견 · 교과 세특
   - 자율·진로·봉사 활동은 **날짜 순(오름차순)**으로 정렬됩니다
3. 각 학생 **왼쪽 칸에 키워드**(성실, 리더십…)를 쉼표로 입력
4. **⚡ 반 전체 초안 생성** → 학생마다 키워드를 반영한 200~300자 문장이 한 번에 채워짐
   - 키워드 안 적은 학생은 **공통 키워드**(상단)로 채워집니다
5. 칸은 길이에 맞춰 **자동으로 늘어나고**, 직접 수정 가능하며 **자동 저장**됩니다
6. 각 행 `🔄`(그 학생만 다시 생성) · `🗑`(그 학생 삭제), 상단 `🗑 일괄 삭제`(전체)
7. `📋 전체 복사` / `⬇️ CSV` 로 NEIS 입력에 활용

### 영역별 특이사항
- **교과 세특**: 반 전체가 아니라 **원하는 학생만 체크**해서 작성합니다(머리글 ☑ = 전체선택). 체크한 학생만 생성·내보내기됩니다.
- **동아리활동**: 반 단위가 아니라 **지원 학생**으로 구성됩니다. 동아리를 누르면 명단이 비어 있고, 왼쪽에서 **학년·반을 옮겨 가며 학생의 `＋추가`** 를 눌러 명단을 만든 뒤 그 학생들에게만 작성합니다. 행의 `✕`로 명단에서 제거합니다. (명단은 동아리별로 따로 저장)

## 주요 기능

- 학생별 **키워드 → 학교 활동 기반 생기부 문장** 자동 생성 (음슴체, 200~300자)
- **반 전체 일괄 생성** (학생마다 표현이 달라짐) · **칸 높이 자동 조절**
- 한글 조사(을/를·이/가) 자동 처리, 활동 **날짜순 정렬**
- 실시간 **글자 수 표시**(200~300자 권장 색상) · 작성 진행률
- **개별/일괄 삭제**, **자동 저장**(브라우저), **백업/복원**, **CSV·전체 복사**
- 상단 **🔒 로컬 / ✨ AI** 토글
  - **로컬**(기본): 무료·오프라인, 인터넷 불필요
  - **AI**: 배포(Vercel) 시 더 자연스러운 문장(기본 모델 **Haiku 4.5**, 1건 약 ₩4).
    학생 이름은 전송하지 않으며, AI 실패 시 자동으로 로컬 생성으로 대체됩니다.

## Vercel 배포 & AI 설정 (✨AI를 쓰려면)

이 앱은 정적 사이트 + 서버리스 함수 1개(`api/generate.js`)로 구성됩니다. AI 생성은 **서버 함수가 환경변수의 API 키로만** Anthropic을 호출하므로 **키가 브라우저에 노출되지 않습니다**.

1. 이 폴더를 GitHub에 올리고 **Vercel**에서 Import (Framework Preset: **Other**, 빌드 설정 불필요)
2. **Settings → Environment Variables** 에 추가:
   | 변수명 | 필수 | 값(예) | 설명 |
   |---|---|---|---|
   | `ANTHROPIC_API_KEY` | ✅ | `sk-ant-...` | Claude API 키 (배포 시 직접 입력) |
   | `ANTHROPIC_MODEL` | 선택 | `claude-haiku-4-5` | 기본 모델 (미설정 시 haiku) |
3. 배포 후 접속 → 상단 **✨ AI** 토글 → 바로 동작 (설정에서 키 입력 불필요)

> - 로컬에서 `index.html`만 열거나 `실행.bat`로 띄우면 **AI는 동작하지 않고 로컬 엔진만** 사용됩니다(서버 함수가 없으므로). 정상입니다.
> - Anthropic 콘솔에서 **월 지출 상한**(예: $60)을 설정하면 비용이 그 금액으로 제한됩니다.
> - ⚠️ **개인정보**: 이 앱은 학생 명단(이름)을 화면 구동을 위해 클라이언트에 포함합니다. 공개 URL이면 명단이 노출될 수 있으니, **Vercel의 Deployment Protection(비밀번호/접근 제한)** 을 켜서 교사만 접근하도록 하는 것을 권장합니다.

## 데이터 출처 (앱에 내장)

- 2026학년도 동화중학교 전체 명단 (657명, 3개 학년 × 8개 반)
- 창의적 체험활동 운영 계획 (예방교육·행사·진로·봉사 일정)
- 동아리 목록 (창체 동아리 + 자율 동아리)
- 교과별 지도 계획 / 학교교육과정 운영계획 (교육목표·교과 활동)

명단을 바꾸려면 `js/students.js` 의 `[학년, 반, 학번, 이름]` 목록을 수정하세요.
활동 문구를 늘리거나 바꾸려면 `js/data.js` 를 수정하면 됩니다.

---

## ⚠️ 유의 사항

생성된 문장은 **초안**입니다. 학생의 실제 활동·관찰 내용을 확인하고
교사가 검토·수정한 뒤 기재하세요. 작성 내용은 **사용 중인 브라우저에만** 저장되므로,
다른 컴퓨터로 옮기거나 학기말 보관 시 **백업** 기능을 이용하세요.

## 폴더 구성
```
index.html         화면
실행.bat           로컬 서버 실행기(Windows, 로컬 엔진만)
api/generate.js    Vercel 서버리스 함수(AI 프록시, 환경변수 키 사용)
.vercelignore      배포 제외 목록(data/, *.bat 등)
css/style.css      디자인
js/students.js     학생 명단(자동 생성 · [학년,반,학번,이름])
js/data.js         학교 활동·교과·키워드 데이터
js/engine.js       로컬 문장 생성 엔진
js/app.js          화면 동작
data/students.json 명단 원본(참고용 · 배포 제외)
```
